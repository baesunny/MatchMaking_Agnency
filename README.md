# 결혼정보회사 매칭 관리 시스템

국민대학교 데이터베이스입문 Term Project — MS Access로 구현한 **인연연구소** 결혼정보회사 고객·매칭·상담 관리 시스템입니다.

## 프로젝트 소개

고객 등록, 선호조건 기반 매칭 추천, 매칭 응답 관리, 상담 이력 기록, 매니저 실적 조회를 하나의 Access DB에서 처리합니다.

| 기능 | 설명 |
|------|------|
| 고객 관리 | 검색, 상세 조회, 신규 등록 |
| 매칭 추천 | 선호·중요 조건 기반 후보 추천 |
| 매칭 관리 | 추천 결과 저장 및 응답·상태 관리 |
| 상담 관리 | 고객별 상담 내용 기록 |
| 매니저 실적 | 담당 매니저별 실적 조회 |
| 고객 포털 | 로그인, 고객 홈, 정보 조회 |

시연 데이터: 남성 100명 · 여성 100명 (고객번호 `D26M001~D26M100`, `D26F001~D26F100`)

## 실행 방법

**요구 사항:** Microsoft Access 2016 이상 (Windows)

1. `database/결정사 엑세스 2_배성윤 차례_3.accdb` 파일을 더블클릭하여 실행합니다.
2. `00_시작화면` 폼이 열리면 원하는 메뉴를 선택합니다.
3. 화면별 사용법은 [`docs/폼_사용_매뉴얼.pdf`](docs/폼_사용_매뉴얼.pdf)를 참고하세요.

## 기본 업무 흐름

1. `00_시작화면` — 메인 메뉴
2. `07_신규고객등록` — 신규 고객 등록
3. `01_고객검색` → `02_고객상세` — 고객 조회
4. `03_매칭추천` — 조건 기반 후보 추천
5. `04_매칭관리` — 매칭 상태·응답 관리
6. `05_상담관리` — 상담 기록
7. `06_매니저실적` — 실적 확인
